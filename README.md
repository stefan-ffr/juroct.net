# juroct.net

Landing page für die Juro-Domain, served via GitHub Pages auf
`juroct.net`.

Custom domain via `CNAME` file. Cloudflare DNS:
- `A juroct.net → 185.199.108-111.153` (GH Pages)
- `AAAA juroct.net → 2606:50c0:8000-8003::153`

Styling via [stefan-ffr/juroct-cdn](https://github.com/stefan-ffr/juroct-cdn) (Juro Corporate Design).
